* Restaurant Ordering Web Application

** Project Overview
This is a full-stack Restaurant Ordering Web Application with Admin and User roles.

1. Features Implemented

 User Authentication
 Admin Authentication
 Add / Edit / Delete Food Items
 Cart System
 Order Placement
 Order Status Tracking
  Manage Stock and Offers

3.
    Technology Stack

  Authentication: JWT
Frontend: React.js
Backend: Node.js + Express
Database: MySQL

5. Setup Instructions

*** Backend
  npm install
  npm start
  Create .env file

### Frontend
1. npm install
2. npm start
